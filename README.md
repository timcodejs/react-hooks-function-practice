# ReactJS Hooks 함수 연습

| List | 적용 | 프로젝트 내용 |
| ------ | -- |----------- |
| useState | ✅ | 카운트 증가 및 감소 상태 관리 & 입력값 추가 상태 관리 |
| useMemo | ✅ | 색상 변경 버튼 클릭 시 결과 반환 후 재사용 |
| useCallback | ✅ | 색상 변경 함수 재사용 |
| useEffect | ✅ | color가 변경 될 때만 결과 반환 |
| useReducer | ✅ | useState로 상태 관리 했던 업데이트 로직을 컴포넌트에서 분리시켜 상태 관리 |
| useContext | ✅ | 전역 상태 관리 |
